$ zarathustra compile -i program.zok --curve bn128

$ zarathustra setup -i out
Setup complete

$ zarathustra compute-witness -i out --abi --stdin <<< '["3", "9"]'
Witness computed

$ zarathustra generate-proof -i out -w witness -p proving.key
✓ Proof generated

$ zarathustra verify -v verification.key -j proof.json

$ zarathustra export-verifier -i verification.key

Everything You Need

A complete zero-knowledge proving toolkit.

📝

High-Level DSL

Python-like syntax with types, generics, structs, arrays, and imports. No low-level constraint wrangling.

🔐

4 Proving Schemes

Groth16, GM17, Marlin (universal), and Nova (IVC). Pick the right scheme for your use case.

⛓️

Solidity Export

One command to generate a verifier contract. Deploy to Ethereum for ~250k gas per verification.

📚

Standard Library

Built-in SHA256, Poseidon, Keccak, MiMC7, Blake2, ECC (BabyJubJub), EdDSA, and SNARK verifiers.

🔄

7 Curves Supported

BN128, BLS12-381, BLS12-377, BW6-761, Pallas, Vesta — covering Ethereum, Filecoin, and Mina ecosystems.

🌐

Web IDE + CLI

Browser-based Monaco editor playground or terminal CLI. Same engine, any interface.
